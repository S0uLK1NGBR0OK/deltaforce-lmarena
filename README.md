# Delta Force LM Arena

This repository contains the structure for Delta Force LM Arena project.

## Structure
- index.html: Main landing page
- about.html, contact.html: Simple info and contact pages
- css/: Stylesheets (includes Bootstrap placeholder)
- js/: JavaScript (custom and dependencies)
- images/: Graphics assets (logos, hero, weapons)
- fonts/: Project font(s)
- libs/: JS libraries (jQuery, GSAP placeholders)
- LICENSE: Optional license
- .gitignore: Standard ignores
